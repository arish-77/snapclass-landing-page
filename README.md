# SnapClass Landing Page

SnapClass is an AI-powered attendance system landing page built to showcase the product, features, workflow, and technology stack.

## Technologies Used

- Python
- Flask
- Gunicorn
- HTML
- CSS
- JavaScript
- Jinja Templates
- Google Fonts
- Vercel Deployment
